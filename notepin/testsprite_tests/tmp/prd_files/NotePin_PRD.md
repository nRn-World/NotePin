\# NotePin - Product Requirements Document



\## Overview

NotePin is a Chrome Extension that allows users to create persistent, floating notes on any webpage. Notes can be "pinned" to specific elements so they follow the scroll and stay in place.



\## Key Features

1\. \*\*Create Notes:\*\* Users can right-click and hold for 1 second on any webpage element to open a context menu and create a new note.

2\. \*\*Persistent Storage:\*\* Notes are saved automatically to the browser's storage and reappear when the user returns to the same page.

3\. \*\*Smart Anchoring:\*\* Notes are attached to specific DOM elements, meaning they move with the content when the user scrolls.

4\. \*\*All Notes View:\*\* A central dashboard where users can see, search, and manage all their notes across different websites.

5\. \*\*Settings:\*\* Users can change languages (Swedish/English), export/import their note data, and customize basic behaviors.

6\. \*\*Premium UI:\*\* Modern glassmorphism design with smooth animations.



\## Target Audience

Researchers, students, and productivity-focused users who need to take quick, contextual notes while browsing the web.
